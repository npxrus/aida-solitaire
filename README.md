# 🃏 Аида — пасьянс «Пирамида»

Классический карточный пасьянс, где нужно собирать пары карт в сумме 13, чтобы очистить пирамиду.

## 🎮 Игровой процесс

- Цель: убрать все карты из пирамиды
- Правила: находите пары карт, которые в сумме дают 13
- Король (13) убирается один
- Туз = 1, Валет = 11, Дама = 12

## 🛠️ Технологии

- [Phaser 4](https://phaser.io/) — игровой фреймворк
- [TypeScript](https://www.typescriptlang.org/) — типобезопасный код
- [Vite](https://vitejs.dev/) — быстрая сборка

## 🚀 Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

## 📦 Сборка

```bash
npm run build
```

Готовая игра появится в папке dist

## 📁 Структура проекта

```text
src/
├── main.ts          # Точка входа, конфигурация Phaser
├── scenes/
│   └── MainScene.ts # Главная игровая сцена
├── entities/
│   ├── Card.ts      # Класс карты
│   └── Deck.ts      # Колода, генерация, перемешивание
└── utils/
    └── constants.ts # Константы (размеры карт, координаты)
```
